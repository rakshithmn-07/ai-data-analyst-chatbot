## ✨ Key Features

- **📁 Instant Uploads**: Seamlessly drag-and-drop your custom `.csv` datasets right into the browser.
- **🧹 Automated Cleaning**: Handle missing data instantly by strictly dropping fields or tactically filling them with averages.
- **🧮 Mathematical Engine**: Compute complex statistics instantaneously (`average`, `max`, `min`, `count`, `group by`).
- **📊 Dynamic Visualization**: The engine automatically scales, labels, and plots gorgeous Histograms, Bar Charts, and Line Graphs.
- **🗺️ Correlation Heatmaps**: Instantly discover deeply correlated variables and uncover hidden dataset relationships.
- **💬 Smart NLP Chatbot**: Type regular, natural questions. The engine will automatically dissect your request, select the best statistical columns to answer it, and bypass errors cleanly.

## 🚀 How to Run Locally

Want to host or modify the chatbot yourself? It takes under two minutes to boot up!

**1. Clone the project**
```bash
git clone https://github.com/[YourUsername]/[YourRepoName].git
cd [YourRepoName]
```

**2. Install all required dependencies**
```bash
pip install -r requirements.txt
```

**3. Launch the dashboard server**
```bash
streamlit run app.py
```
